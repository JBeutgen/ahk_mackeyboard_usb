# ahk_mackeyboard_usb
AutoHotKey Script for USB Mackeyboard
Allows usage of mac keyboard on Windows with Windows-keystyle 
- left alt = left windows
- left cmd = left alt
- right cmd = alt gr
- right alt = fn
- remapped all the media function keys

# usage
- Download AutoHotKey from https://autohotkey.com/
- Execute .ahk file
- Optional: put into autorun to start with machine
- Script can be toggled on/off with ctrl + f12 (useful if on notebook)
